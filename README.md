## Terminal Tetris
Tetris running in a unix terminal, using ncurses

### Setup
1. Install ncurses if needed
2. `cd` to main source directory
3. `mkdir build`
4. `cd build`
5. `cmake ..`
6. `make`
7. `./tetris` to run

### Controls
- `HJKL` to move and navigate menus
- `A` and `S` to rotate
- `D` to store a piece
